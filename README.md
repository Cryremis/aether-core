# AetherCore

AetherCore 是一个独立于业务平台的通用 Agent Runtime 平台。

当前仓库包含三个子项目：

- `backend`：独立后端服务，负责会话、宿主注入、文件与技能管理、运行时编排、沙箱抽象。
- `frontend`：独立工作台前端，面向用户呈现 Agent 执行过程、文件、技能与结果。
- `host-adapters/dash`：面向 `ascend-compete-dash` 的宿主接入适配器骨架。

## 设计原则

- 与宿主平台解耦：平台通过宿主注入协议传递上下文、工具、技能与 API 描述。
- 执行强隔离：文件、技能与后续脚本执行均落在会话工作区与沙箱抽象中。
- 协议优先：前后端、宿主与运行时之间统一走结构化协议，便于扩展与治理。
- 渐进建设：先打通最小闭环，再持续增强 LLM、Tool、Skill、Sandbox 与 Artifact 能力。

## 当前阶段

当前版本完成第一版工程骨架，重点落地：

- 独立后端服务与 `/api/v1` 路由结构
- 宿主注册与会话绑定协议
- 会话工作区、文件上传与产物清单服务
- 技能注册服务与内置技能占位
- 运行时事件协议与 SSE 工作流骨架
- 独立工作台前端基础结构
- Dash 宿主适配器初版说明与接入骨架
